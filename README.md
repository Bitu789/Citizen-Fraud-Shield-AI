# FraudShield AI

## Problem Statement

Online users frequently become victims of phishing attacks, OTP scams, fake banking alerts, lottery fraud, and investment scams.

## Solution

FraudShield AI is a web application that analyzes suspicious messages and provides:

* Fraud Risk Score
* Risk Level Classification
* Fraud Type Detection
* Explainable Detection Reasons

## Features

* Banking Scam Detection
* OTP Scam Detection
* Lottery Scam Detection
* Investment Scam Detection
* Real-Time Analysis
* React Frontend
* FastAPI Backend

## Technology Stack

Frontend:

* React
* Vite

Backend:

* FastAPI
* Python

## How It Works

User Message
→ Fraud Analysis Engine
→ Risk Classification
→ Fraud Report
